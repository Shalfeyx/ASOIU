# Restaurant Management App

ASP.NET Core MVC приложение для управления ресторанами и блюдами.

## Функциональность
- CRUD операции для ресторанов (с фото)
- CRUD операции для блюд (с валидацией цены)
- Отчёты (3 раздела) через LINQ
- Защита от удаления ресторанов с блюдами

## Технологии
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core (SQLite)
- Bootstrap 5
- LINQ to Entities
