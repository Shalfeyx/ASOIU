# Домашнее задание N2 - Консольное приложение "Управление ресторанами и блюдами"

## Описание проекта

Консольное приложение на C# для управления ресторанами и блюдами с использованием SQLite и паттерна Fluent Interface для построения отчётов.

## Вариант задания

Вариант 22: Рестораны (справочник) и Блюда (основная таблица)

- Справочная таблица: рестораны (ID, название)
- Основная таблица: блюда (ID, ID ресторана, название, цена)
- Числовое поле: цена блюда (не может быть отрицательной)

## Технологии

- C# (.NET 8)
- SQLite (Microsoft.Data.Sqlite)
- ADO.NET для работы с БД
- Паттерн Fluent Interface для отчётов
